## 一、格式验证

- **通过**：是 / 否

## 附录：特殊情况处理

### 情况1：验证进行中

如果验证仍在进行中，报告应简化为：

```markdown
# Skill 验证报告

## 基本信息

| 字段 | 值 |
|------|-----|
| **Skill ID** | xxx |
| **名称** | xxx |
| **验证阶段** | layer1 / layer2 |

## 说明

验证正在进行中（第一层/第二层验证），请稍后刷新查看完整报告。
```

### 情况2：格式验证失败

如果格式验证失败，报告应简化为：

```markdown
# Skill 验证报告

## 基本信息

| 字段 | 值 |
|------|-----|
| **Skill ID** | xxx |
| **名称** | xxx |
| **验证阶段** | failed |

## 一、格式验证

- **通过**：否
- **错误**：
  - Missing SKILL.md
  - Invalid frontmatter format

## 结论

**验证状态**：❌ 失败

**原因**：Skill 格式验证失败，请修正后重新上传。
```

### 情况3：第一层验证失败

如果第一层验证失败但评分不是极低，应提供完整的失败报告。

### 情况4：回归验证失败

如果回归验证失败，应重点说明哪些已入库的 Skill 受到影响，以及可能的原因。
